# StoryFi 子网部署检查清单

## ⚠️ 关键问题澄清

在开始部署之前，你**必须**先了解以下信息：

---

## 1. 你朋友的子网当前状态是什么？

### 情况 A：全新空子网（最理想）

**特征**：
- ✅ 子网已注册（有 netuid）
- ✅ Alpha 代币已发行
- ❌ **没有**运行任何 Validator
- ❌ **没有**运行任何 Miner
- ❌ **没有**现有的激励机制代码

**如果是这种情况**：
- ✅ **不需要**与 Bittensor 官方沟通
- ✅ **不需要**官方批准
- ✅ 直接部署你的代码即可
- ✅ 你是第一个 Validator，获得 100% Validator 排放

**需要做的**：
1. 部署 Validator（你的 `validator.py`）
2. 部署至少 1 个 Miner（你的 `miner.py`）
3. 等待其他 Miners 加入
4. 开始获得 TAO 排放

---

### 情况 B：已有其他机制运行中（需要替换）

**特征**：
- ✅ 子网已注册
- ✅ 已经有 Validator 在运行
- ✅ 已经有 Miners 在运行
- ✅ 已经有现有的激励机制

**如果是这种情况**：
- ⚠️ **需要慎重考虑**
- ⚠️ 替换机制会影响现有参与者
- ⚠️ 可能导致现有 Miners 失效
- ⚠️ 需要与现有参与者沟通

**需要做的**：
1. **停止旧的 Validator**
2. **部署新的 Validator**（你的代码）
3. **通知所有 Miners** 更新代码
4. **给予过渡期**（例如 1 周）

---

### 情况 C：子网还没创建（最复杂）

**特征**：
- ❌ 子网还没注册
- ❌ 需要从零开始

**如果是这种情况**：
- ⚠️ **需要大量 TAO**（数千到数万）
- ⚠️ **需要与 Bittensor 社区沟通**
- ⚠️ **可能需要提案和投票**

**费用估算**（2025年数据）：
- 注册费用：**动态定价**，取决于当前子网数量
- 通常：**1,000 - 10,000 TAO**（$500K - $5M）
- 这些 TAO 会被**永久锁定（burned）**

**流程**：
1. 准备充足的 TAO
2. 在测试网测试至少 1 周
3. 准备技术文档和白皮书
4. （可选）在社区发起讨论
5. 使用 `btcli subnet create` 注册
6. 等待 7 天免疫期
7. 开始获得排放

## 3. 是否需要与 Bittensor 官方沟通？

### ✅ 不需要官方沟通的情况

1. **在已有子网上部署代码**
   - 子网已经注册好
   - 你只是作为 Validator/Miner 加入
   - **不需要**官方批准

2. **修改子网的激励机制**
   - 子网 Owner 有权修改代码
   - **不需要**官方批准
   - 但需要考虑现有参与者

3. **运行自己的 Validator**
   - 只需注册 hotkey（~1 TAO）
   - **不需要**官方批准

---

### ⚠️ 需要官方沟通的情况

1. **创建全新子网**
   - 需要锁定大量 TAO
   - 建议先在 Discord 咨询
   - 了解当前注册费用

2. **子网参数重大变更**
   - 例如修改排放比例（18% / 41% / 41%）
   - 这需要链上治理
   - 需要社区投票

3. **遇到技术问题**
   - 例如无法注册、无法上链
   - 可以在 Discord 求助

## 4. dTAO 机制下的关键点

### Alpha 代币经济学

**每个子网都有自己的 Alpha 代币**：

```
子网 A → Alpha_A
子网 B → Alpha_B
子网 C → Alpha_C
```

**TAO 排放取决于 Alpha 代币价格**：

```
Alpha 价格 ↑ → 更多 TAO 流入 → 更高排放
Alpha 价格 ↓ → 更少 TAO 流入 → 更低排放
```

**流动性池结构**：

```
TAO-Alpha_A Pool
├── TAO（来自 TAO 排放的 50%）
└── Alpha_A（来自新发行的 50%）
```

**排放分配**：

```
新发行的 Alpha 代币：
├── 50% → 流动性池
└── 50% → 分配给参与者
    ├── 18% → Subnet Owner
    ├── 41% → Validators
    └── 41% → Miners
```

## 7. 不需要官方批准的原因

### Bittensor 的去中心化设计

1. **任何人都可以注册 Validator/Miner**
   - 只需支付注册费（~1 TAO）
   - 无需许可

2. **Subnet Owner 拥有完全控制权**
   - 可以随时修改激励机制代码
   - 可以自己运行 Validator
   - 无需官方批准

3. **市场机制决定成功**
   - Alpha 代币价格 = 市场评价
   - 不是官方决定，而是 TAO 持有者决定
